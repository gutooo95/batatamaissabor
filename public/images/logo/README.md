# Pasta de Logo

Coloque aqui o arquivo do logo da empresa.

## Nome do arquivo sugerido:
- `logo.png` ou `logo.svg`

O código está configurado para procurar por `logo.png` na pasta `/images/logo/`.

Se você usar outro nome, atualize o arquivo `components/Header.tsx` na linha que contém:
```tsx
src="/images/logo/logo.png"
```

## Formatos suportados:
- PNG (recomendado)
- SVG (recomendado para melhor qualidade)
- JPG/JPEG
