# Pasta de Imagens da Frota

Coloque aqui as imagens da frota de caminhões para a seção "Nossa Frota".

## Arquivo necessário:

### Imagem principal:
- **Nome**: `caminhao-frota.jpg` (ou `.png`)
- **Tamanho recomendado**: 1920x1080px ou maior
- **Uso**: Imagem cinematográfica do caminhão em movimento na estrada ao pôr do sol
- **Efeito**: A imagem será exibida com filtros de brilho, contraste e saturação para criar um efeito cinematográfico

## Especificações recomendadas:
- **Formato**: JPG (recomendado) ou PNG
- **Tamanho**: Otimize para web (recomendado: < 500KB)
- **Conteúdo**: Foto de um caminhão da frota em movimento, preferencialmente em uma estrada ao pôr do sol
- **Efeito visual**: Motion blur ou movimento para transmitir agilidade

## Nota:
Se a imagem não for encontrada, o sistema exibirá um placeholder automaticamente.
