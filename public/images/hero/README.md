# Pasta de Imagens do Hero

Coloque aqui a imagem de fundo da seção Hero (primeira seção do site).

## Nome do arquivo sugerido:
- `hero-background.jpg` ou `hero-background.png`

O código está configurado para procurar por `hero-background.jpg` na pasta `/images/hero/`.

## Especificações recomendadas:
- **Resolução**: 1920x1080px ou maior (Full HD)
- **Formato**: JPG (recomendado) ou PNG
- **Tamanho**: Otimize para web (máximo 500KB recomendado)
- **Conteúdo**: Imagem relacionada a batatas, produtos, ou tema do negócio

## Nota:
Se a imagem não for encontrada, o sistema usará um gradiente de cores como fallback.
