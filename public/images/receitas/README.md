# Pasta de Imagens de Receitas

Coloque aqui as imagens apetitosas das receitas para os cards da seção "Dicas do Chef".

## Arquivos necessários:

1. **`estrogonofe-de-carne.jpg`** - Para a receita "Estrogonofe de Carne"
2. **`estrogonofe-de-frango.jpg`** - Para a receita "Estrogonofe de Frango"

## Especificações recomendadas:

- **Formato**: JPG ou WebP
- **Tamanho**: 800x600px ou 1200x800px
- **Qualidade**: Otimizada (80-85)
- **Conteúdo**: Imagens apetitosas e profissionais das receitas prontas
- **Estilo**: Comida bem fotografada, boa iluminação, cores vibrantes

## Nota:

Se as imagens não forem encontradas, o sistema usará imagens placeholder automaticamente.

## Dica:

Use imagens que:
- Mostrem o prato finalizado de forma apetitosa
- Tenham boa composição e iluminação
- Destacem a batata palha no prato
- Sejam otimizadas para web (< 200KB recomendado)
