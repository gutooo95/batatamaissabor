# Pasta de Imagens de Lavouras

Coloque aqui as imagens de lavouras de batatas para a seção "Nossa Origem, Nosso Orgulho".

## Arquivos necessários:

### Imagem de fundo (parallax):
- **Nome**: `lavoura-background.jpg` ou `lavoura-background.png`
- **Tamanho recomendado**: 1920x1080px ou maior
- **Uso**: Imagem de fundo com efeito parallax na seção

### Imagens da galeria (4 imagens):
1. **`lavoura-1.jpg`** - Imagem vertical (ex: plantio, campo)
2. **`lavoura-2.jpg`** - Imagem quadrada (ex: qualidade, processo)
3. **`lavoura-3.jpg`** - Imagem quadrada (ex: colheita, trabalho)
4. **`lavoura-4.jpg`** - Imagem vertical (ex: lavoura, paisagem)

## Especificações recomendadas:
- **Formato**: JPG (recomendado) ou PNG
- **Tamanho**: Otimize para web
- **Conteúdo**: Fotos reais de lavouras, plantio, colheita, ou processos relacionados à produção de batatas

## Layout:
As imagens serão exibidas em um grid 2x2:
- **Coluna esquerda**: lavoura-1 (vertical) e lavoura-2 (quadrada)
- **Coluna direita**: lavoura-3 (quadrada) e lavoura-4 (vertical)

## Nota:
Se alguma imagem não for encontrada, o sistema usará imagens placeholder automaticamente.
