# Pasta de Embalagens

Coloque aqui as fotos das embalagens dos produtos.

## Nomes dos arquivos sugeridos:

1. **Batata Ondulada - Original**: `batata-ondulada-original.jpg`
2. **Batata Ondulada - Churrasco**: `batata-ondulada-churrasco.jpg`
3. **Batata Ondulada - Cebola e Salsa**: `batata-ondulada-cebola-salsa.jpg`
4. **Batata Palha - Tradicional**: `batata-palha-tradicional.jpg`
5. **Batata Palha - Premium**: `batata-palha-premium.jpg`

## Como funciona:

O código está configurado para usar essas imagens automaticamente. Se você usar nomes diferentes, atualize o arquivo `constants.tsx` com os nomes corretos.

## Formatos suportados:
- JPG/JPEG (recomendado)
- PNG
- WebP

## Dica:
- Use imagens com boa qualidade (mínimo 400x500px recomendado)
- As imagens serão redimensionadas automaticamente pelo navegador
- Se uma imagem não for encontrada, o sistema usará uma imagem placeholder automaticamente
