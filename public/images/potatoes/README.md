# Pasta de Imagens de Batatas

Coloque aqui as imagens individuais de batatas (PNGs) para o efeito de explosão na Hero Section.

## Arquivos necessários:

- `potato-1.png` até `potato-5.png` (mínimo 5 imagens, serão repetidas para criar 20 batatas)

## Especificações recomendadas:

- **Formato**: PNG com fundo transparente
- **Tamanho**: 200x200px a 400x400px
- **Estilo**: Batatas individuais, diferentes ângulos/posições
- **Qualidade**: Otimizadas (WebP ou PNG comprimido)

## Nota:

Se as imagens não forem encontradas, o sistema usará SVGs como fallback automaticamente.

## Dica:

Você pode usar:
- Batatas em diferentes ângulos
- Batatas inteiras e cortadas
- Diferentes tipos de batatas
- Variações de cor/iluminação
